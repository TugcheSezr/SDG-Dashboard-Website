## 📌 Over Het Project

SDG Dashboard is een full-stack webapplicatie die alle 17 Sustainable Development Goals overzichtelijk en interactief visualiseert.  

Focuspunten:
- Clean UI/UX 🎨  
- Data visualisatie 📊  
- Schaalbare architectuur 🏗️  
- Praktijkgerichte development skills 💻  

---

## ⚡ Features

- 🌍 Overzicht van alle 17 SDG’s  
- 📊 Interactieve grafieken met Chart.js  
- 🔎 Detailpagina’s met inzichten  
- ⚡ Supersnelle performance met Next.js  
- 🎨 Responsive en moderne UI  
- 🧠 Type-safe development met TypeScript  
- 🗄️ Prisma + MySQL integratie  
- 📦 Schaalbare projectstructuur  

---

## 🛠️ Tech Stack

**Frontend**
- Next.js 14  
- React  
- TypeScript  
- Tailwind CSS  

**Backend**
- Next.js API Routes  
- Prisma ORM  
- MySQL  

**Data Visualisatie**
- Chart.js

## 🚀 Installatie & Starten

```bash
# Repo clonen
git clone https://github.com/TugcheSezr/SDG-Dashboard-Website.git

# Naar project map
cd SDG-Dashboard-Website

# Dependencies installeren
npm install

# Environment setup
cp .env.sample .env

# Database setup
npm run db:generate
npm run db:push
npm run db:seed

# Development server starten
npm run dev

🌐 Open in browser:
http://localhost:3000
